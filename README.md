# Blinkit-data-analysis-project

This project analyzes Blinkit sales data to understand product demand, outlet performance, and sales trends.  
The analysis helps identify top-selling items, customer preferences, and factors affecting sales.

**Tools Used:** Python, Pandas, NumPy, Matplotlib, Seaborn  
**Key Tasks:** Data cleaning, EDA, sales analysis, visualization
